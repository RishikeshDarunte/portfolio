# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/RishikeshDarunte/portfolio
```

Navigate to the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm start
```

For Vite projects

```bash
npm run dev
```
